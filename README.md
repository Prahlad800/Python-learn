# Python Learn

A complete beginner-to-advanced Python learning project.

The course is organized chapter by chapter under
`Section-01-Python-Fundamentals`. Each `.py` lesson includes:

- Topic name and theory
- Syntax notes
- Multiple runnable examples
- Practice programs
- A mini-project prompt
- Interview questions with answers
- Sample expected output in comments

## How to Run

```bash
cd Section-01-Python-Fundamentals/Chapter-01-Introduction
python introduction.py
```

You can run any lesson file directly. The examples avoid external
dependencies and are designed for Python 3.10+.
