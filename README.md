## 🗺️ Learning Path & Folder Structure

The course is structured logically into distinct phases:

### Phase 1: Core Fundamentals (`Section-01-Python-Fundamentals`)
- `Chapter-01-Introduction` - Python history, setup, syntax basics.
- `Chapter-02-Variables-and-DataTypes` - Strings, Numbers, Booleans, NoneType.
- `Chapter-03-Operators` - Arithmetic, Logical, Bitwise, Walrus, Precedence.
- `Chapter-04-Input-Output` - f-strings, formatting, user input validation.
- `Chapter-05-Conditional-Statements` - if/else, match-case, guard clauses.
- `Chapter-06-Loops` - for, while, comprehensions, infinite loops.
- `Chapter-07-Functions` - arguments, recursion, closures, lambdas.
- `Chapter-08-Strings` - operations, regex intro, encoding, validation.
- `Chapter-09-Lists` - slicing, sorting, stacks/queues, comprehensions.
- `Chapter-10-Tuples` - packing/unpacking, named tuples.
- `Chapter-11-Dictionaries` - methods, defaultdict, merging patterns.
- `Chapter-12-Sets` - frozensets, set algorithms, performance.
- `Chapter-13-Modules-Packages` - imports, PIP, virtual environments.
- `Chapter-14-Exception-Handling` - try/except, custom exceptions, chaining.
- `Chapter-15-File-Handling` - read/write, context managers, pathlib, CSV/JSON.

### Phase 2: Object-Oriented & Advanced Features
- `Chapter-16-OOP` - Classes, Inheritance, Polymorphism, Encapsulation, Dataclasses.
- `Chapter-17-Decorators` - function/class decorators, wraps, caching.
- `Chapter-18-Generators` - yield, memory efficiency, pipelines.
- `Chapter-19-Iterators` - iter/next protocols, itertools.
- `Chapter-20-Multithreading` - GIL, thread pools, locks, synchronization.
- `Chapter-21-Asyncio` - async/await, event loops, tasks.

### Phase 3: Intermediate & Standard Library
*(Additional chapters covering Regex, Collections, Typing, Context Managers, Logging, Testing with Pytest/Unittest, SQLite, Networking).*

### Phase 4: Advanced Python & Software Architecture
*(Chapters covering Design Patterns, Data Structures, Algorithms, Metaclasses, Concurrency/Parallelism, Clean Code Principles).*

### Phase 5: Real-World Projects (`projects/`)
Contains standalone mini-projects to integrate your learning:
- Calculator
- Bank System
- Student Management
- File Organizer
- Weather App
- REST API (Flask/FastAPI)
- And more...

## 🚀 How to Run the Code

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Prahlad800/Python-learn.git
   cd Python-learn
   ```

2. **Navigate to a chapter:**
   ```bash
   cd Section-01-Python-Fundamentals/Chapter-01-Introduction
   ```

3. **Run a specific file:**
   ```bash
   python first_program.py
   ```
   *Note: Every file contains a `main()` block. We highly encourage you to open the file in your IDE (VSCode, PyCharm) to read the theory and modify the code as you learn!*
